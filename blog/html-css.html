What does it mean to display inline vs inline blocks?

5 September 2023

Within CSS, there are many ways to display HTML information. One of these ways is by using the display property to show content as a 'block', 'inline', or combining properties of both into 'inline-block'.

Block

- allows the user to modify the height and width of an element, however, it will create a line break (so that no elements are on the same line).

This what I have written in CSS to make the text look as it does below:

Te Houtaewa was the fastest runner of his day and played many pranks on his peoples enemies. One morning his mother wanted kumara for the hangi (earth oven) and asked Te Houtaewa to go to the gardens at Te Kao, a short distance away.

Inline

- does not allow the user to modify the height and width of an element, however, it will allow all elements to stay on the same line.

This what I have written in CSS to make the text look as it does below:

Instead of going to the nearby gardens, Te Houtaewa set off for Ahipara as he wanted to annoy some Te Rarawa people who lived there.

Inline-block

- allows the user to modify the height and width of an element, however, it will allow all elements to stay on the same line. Thus combining elements of both the named properties.

This what I have written in CSS to make the text look as it does below:

While Te Houtaewa was filling his baskets with kumara he was spotted stealing the kai (food). He was immediately recognised.
